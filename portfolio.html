personal portfolio website

this website is my first REAL programming project! the code is almost entirely written by me (aside from some parts where i didn't know how to do things and consulted copilot, not my strongest moment) and it is my first fully functional website.

all the code in this site is open source on github (in fact this site is literally hosted from github. lol) and written in vscode. i'm sure if a professional looked at the code for this website and maybe even the organization of the repo they would be infuriated, but then again this is my first project, so it's obviously not the best.

some notable features and methods i used:

  • all images hand edited by me.
    • the animated logo was made in pixelmator pro and then animated in apple motion
    • my photography work is edited in a mix of lightroom and pixelmator
    • the covers for my plays were made in adobe express
  • single, centralized css stylesheet (probably common, but in my web design class we make individual style elements in the headers of each page)
  • multiple page layout with a static header!
  • fully functional contact form made with formsubmit

for my first ever web dev and programing project, i'm quite proud of this. i plan on expanding to ios apps and more complex web design, and this site will be the hub for all of my creative work.

thanks for checking my portfolio out! :)
